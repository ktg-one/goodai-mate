<threat_model>
## Threat Model

- T-02-04: Client component accidentally exposes `AI_GATEWAY_API_KEY`.
  - Mitigation: call only `/api/chat`; never read server-only env vars in client code.
- T-02-05: API errors leak stack traces to visitors.
  - Mitigation: catch fetch/stream errors and render short friendly copy.
- T-02-06: User-entered transcript is sent to lead form unexpectedly.
  - Mitigation: only pass transcript to `LeadCaptureCard` after the first assistant response and keep behavior visible.
</threat_model>
