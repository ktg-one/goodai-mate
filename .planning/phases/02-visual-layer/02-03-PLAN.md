  <name>Task 1: Create CursorEffects component with cursor, glow, hover, and touch detection</name>
  <files>src/components/CursorEffects.tsx</files>
  <action>
    **CursorEffects.tsx** — Create a `'use client'` component that renders the custom cursor and ambient glow:

    1. **Touch detection (early return):**
       - On mount, check: `'ontouchstart' in window || navigator.maxTouchPoints > 0`
       - If touch device: return `null` (render nothing). Set `document.body.style.cursor = 'auto'`.
       - If NOT touch device: set `document.body.style.cursor = 'none'` to hide native cursor.

    2. **Render two elements:**
       ```tsx
       <>
         {/* Custom cursor */}
         <div
           ref={cursorRef}
           className="fixed z-[9999] pointer-events-none rounded-full mix-blend-difference"
           style={{
             width: '20px',
             height: '20px',
             border: '1.5px solid rgba(216,106,61,0.35)',
             transform: 'translate(-50%, -50%)',
             transition: 'width 0.3s ease, height 0.3s ease, border-color 0.3s ease',
             left: 0,
             top: 0,
           }}
         />
         {/* Ambient glow */}
         <div
           ref={glowRef}
           className="fixed z-0 pointer-events-none rounded-full"
           style={{
             width: '700px',
             height: '700px',
             background: 'radial-gradient(circle, var(--accent-glow) 0%, transparent 65%)',
             transform: 'translate(-350px, -350px)',
             transition: 'left 0.9s cubic-bezier(0.23, 1, 0.32, 1), top 0.9s cubic-bezier(0.23, 1, 0.32, 1)',
             left: 0,
             top: 0,
           }}
         />
       </>
       ```

    3. **Mouse tracking with lerp (requestAnimationFrame):**
       - Store raw mouse position in `mouseRef = useRef({ x: 0, y: 0 })`
       - Store current cursor position in `cursorPos = useRef({ x: 0, y: 0 })`
       - `pointermove` listener on `window`: update `mouseRef.current`
       - RAF loop:
         ```ts
         const lerp = 0.12;
         cursorPos.current.x += (mouseRef.current.x - cursorPos.current.x) * lerp;
         cursorPos.current.y += (mouseRef.current.y - cursorPos.current.y) * lerp;
         if (cursorRef.current) {
           cursorRef.current.style.left = `${cursorPos.current.x}px`;
           cursorRef.current.style.top = `${cursorPos.current.y}px`;
         }
         ```
       - Glow follows mouse directly (CSS transition handles easing):
         ```ts
         if (glowRef.current) {
           glowRef.current.style.left = `${mouseRef.current.x}px`;
           glowRef.current.style.top = `${mouseRef.current.y}px`;
         }
         ```

    4. **Hover detection with event delegation (CURS-03, CURS-06):**
       - Use event delegation on `document` instead of binding to individual elements. This automatically handles dynamically injected elements (chat messages, lead card) without re-registration:
         ```ts
         const handleMouseOver = (e: MouseEvent) => {
           const target = (e.target as HTMLElement).closest('input, button, a, [data-cursor-hover]');
           if (target && cursorRef.current) {
             cursorRef.current.style.width = '48px';
             cursorRef.current.style.height = '48px';
             cursorRef.current.style.borderColor = 'rgba(216,106,61,0.55)';
           }
         };
         const handleMouseOut = (e: MouseEvent) => {
           const target = (e.target as HTMLElement).closest('input, button, a, [data-cursor-hover]');
           if (target && cursorRef.current) {
             cursorRef.current.style.width = '20px';
             cursorRef.current.style.height = '20px';
             cursorRef.current.style.borderColor = 'rgba(216,106,61,0.35)';
           }
         };
         document.addEventListener('mouseover', handleMouseOver);
         document.addEventListener('mouseout', handleMouseOut);
         ```
       - Event delegation on `document` means ANY new `input`, `button`, `a`, or `[data-cursor-hover]` element injected into the DOM will automatically get hover behavior — no re-registration needed (solves CURS-06 and Pitfall 9).

    5. **Cleanup (useEffect return):**
       - Cancel RAF
       - Remove `pointermove`, `mouseover`, `mouseout` listeners
       - Restore `document.body.style.cursor = 'auto'`

    6. **Export a function `registerCursorTargets` as a named export** for Phase 3/4 to call if needed — but with event delegation, this is effectively a no-op. Include it as a safety valve:
       ```ts
       export function registerCursorTargets() {
         // Event delegation handles this automatically.
         // This function exists as API contract for Phase 3/4.
       }
       ```
