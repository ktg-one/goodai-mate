  <name>Task 2: Create ShaderBackground component with Three.js and wire into page.tsx</name>
  <files>src/components/ShaderBackground.tsx, src/app/page.tsx</files>
  <action>
    **ShaderBackground.tsx** — Create a `'use client'` component that sets up a vanilla Three.js scene inside a `useEffect`:

    1. **Imports:** `import * as THREE from 'three'` and `import fragmentShader from '@/shaders/sdf-lensblur.frag'`.

    2. **Component structure:**
       - `containerRef = useRef<HTMLDivElement>(null)`
       - Return `<div ref={containerRef} className="fixed inset-0 -z-10" aria-hidden="true" />`

    3. **WebGL detection:** Before creating renderer, check `typeof WebGLRenderingContext !== 'undefined'` and attempt to create a test canvas context. If WebGL unavailable, set a state flag and render the fallback div instead:
       ```tsx
       <div className="fixed inset-0 -z-10" style={{
         background: 'linear-gradient(135deg, #0C0C0C 0%, #1a1410 50%, #0C0C0C 100%)'
       }} aria-hidden="true" />
       ```

    4. **Three.js setup inside useEffect (only runs if WebGL available):**
       - `renderer = new THREE.WebGLRenderer({ antialias: false, alpha: true })`
       - Append `renderer.domElement` to `containerRef.current`
       - `scene = new THREE.Scene()`
       - `camera = new THREE.OrthographicCamera()` — bounds set in resize handler
       - `camera.position.z = 1`
       - `geo = new THREE.PlaneGeometry(1, 1)`
       - Mouse vectors: `vMouse = new THREE.Vector2()`, `vMouseDamp = new THREE.Vector2()`, `vResolution = new THREE.Vector2()`
       - `mat = new THREE.ShaderMaterial` with:
         - Inline vertex shader (passthrough): `varying vec2 v_texcoord; void main() { gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0); v_texcoord = uv; }`
         - `fragmentShader` imported from .frag file
         - `uniforms: { u_mouse: { value: vMouseDamp }, u_resolution: { value: vResolution }, u_pixelRatio: { value: 2 } }`
         - `defines: { VAR: 0 }`
       - `quad = new THREE.Mesh(geo, mat)`, add to scene

    5. **Mouse tracking:**
       - `pointermove` listener on `window`: `vMouse.set(e.pageX, e.pageY)`
       - `touchmove` on `document.body` with `{ passive: false }` and `e.preventDefault()` to prevent scroll conflict

    6. **Resize handler:**
       - `w = window.innerWidth; h = window.innerHeight`
       - `dpr = Math.min(window.devicePixelRatio, 2)` (SHDR-06: cap at 2, further reduce on mobile: if `window.innerWidth < 640`, cap at 1)
       - `renderer.setSize(w, h); renderer.setPixelRatio(dpr)`
       - Camera bounds: `left=-w/2, right=w/2, top=h/2, bottom=-h/2`, then `updateProjectionMatrix()`
       - `quad.scale.set(w, h, 1)`
       - `vResolution.set(w, h).multiplyScalar(dpr)`
       - `mat.uniforms.u_pixelRatio.value = dpr`
       - Call resize immediately after setup

    7. **Animation loop:**
       - `requestAnimationFrame` loop
       - Delta time from `performance.now() * 0.001`
       - Mouse damping per frame: `THREE.MathUtils.damp(vMouseDamp.x, vMouse.x, 8, dt)` and same for y
       - `renderer.render(scene, camera)`

    8. **Cleanup (useEffect return):**
       - Cancel RAF via `cancelAnimationFrame(rafId)`
       - Remove `pointermove` and `touchmove` listeners
       - Remove `resize` listener
       - `geo.dispose(); mat.dispose(); renderer.dispose(); renderer.forceContextLoss()`
       - Remove renderer.domElement from container
       - This handles React 19 Strict Mode double-mount in dev (Pitfall 3)

    **page.tsx** — Update the root page to dynamically import ShaderBackground:

    ```tsx

    const ShaderBackground = dynamic(() => import('@/components/ShaderBackground'), {

    });

        <>

        </>
      );
    }
    ```

    Ensure page.tsx remains a Server Component (no 'use client'). ShaderBackground is the only client island at this point.
