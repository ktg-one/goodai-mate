**Props:**
```typescript

```

**State:**
```typescript
const [messages, setMessages] = useState<{ role: 'user' | 'ai'; content: string }[]>([]);
const [isWaiting, setIsWaiting] = useState(false);
const [inputValue, setInputValue] = useState('');

const hasReceivedFirstResponse = useRef(false);
```

**On mount (useEffect):**
When component mounts with `initialMessage`, immediately:
1. Add user message to messages array
2. Call `sendToAPI([{ role: 'user', content: initialMessage }])`

**`sendToAPI(messagesToSend)` function:**
1. Set `isWaiting = true` (CHAT-08)
2. `fetch('/api/chat', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ messages: messagesToSend }) })`
3. Parse response JSON
4. Add AI message to messages array: `{ role: 'ai', content: data.response }`
5. Set `isWaiting = false`
6. If this is the first AI response and `onFirstResponse` exists, call it
7. On fetch error: add AI message with "Something broke on our end. Try again in a moment."

**`handleSubmit(text)` function:**
1. Guard: if empty or `isWaiting`, return
2. Add user message to messages
3. Build full messages array (all messages + new one)
4. Clear input
5. Call `sendToAPI(allMessages)`
